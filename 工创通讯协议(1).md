### TYPE
数据类型：内容，反馈这些
```cpp
enum MsgType : uint8_t {
    TYPE_CMD       = 0x01,  // 上位机发给下位机的控制命令
    TYPE_ACK       = 0x02,  // 下位机对命令的接收应答
    #TYPE_STATUS    = 0x03,  // 下位机周期状态上报
    #TYPE_EVENT     = 0x04,  // 下位机动作完成事件
    #TYPE_HEARTBEAT = 0x05,  // 心跳包
    #TYPE_ERROR     = 0x06   // 错误上报
};
```
